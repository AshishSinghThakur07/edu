# EduManagement App

A comprehensive MERN stack application for managing educational institutions.

## Features
- **Roles**: Student, Teacher, HR, Admin
- **Modules**:
    - Institution Management
    - User Profiles
    - Course & Class Management
    - Timetable & Attendance
    - Assignments & Submissions
    - Job Postings & Applications
- **Tech Stack**: MongoDB, Express, React, Node.js
- **PWA**: Installable on devices.

## Setup

### Backend
1. `cd server`
2. `npm install`
3. Create `.env` file (see `.env.example` or code)
4. `npm run dev` (starts on port 5000)

### Frontend
1. `cd client`
2. `npm install`
3. `npm run dev` (starts on port 5173)
